## Overview
The purpose of the website is to give customers and potential customers key information about the business including what they can expect, hours of operation, event dates and times, and the perks of joining the Wine Club.

## Events
The Events section is for planned events that do not influence the store's opening hours. When an event is created, it gets added to the Calendar and Upcoming Events list. Events that have passed still appear on the calendar but disappear from the Upcoming Events list. Clicking or tapping on an event preview on the calendar will display a popup with the full event details.
### Event Fields
- Title
- Subtitle (optional)
- Date
- Start Time
- End Time
- Description (optional)
- Registration Link (optional)
#### Title
The full event title. This is what displays in the event popup and in the Upcoming Events list. Titles should be just long enough to let people know what the event is at a glance.
#### Subtitle
The subtitle, if added, is what displays on the calendar. The subtitle should be even shorter than the title and let people know the type of event.
#### Date
The date of the event. This tells the site where to place the event on the calendar and how to organize the event in the Upcoming Events list. The date is displayed on the event popup and in the Upcoming Events list.
#### Start Time
The time the event begins. The start time is displayed on the event popup, in the Upcoming Events list, and on the calendar as the subtitle if no subtitle was specified. When two events are scheduled for the same day, the events will be sorted by start time on the calendar and Upcoming Events list.
#### End Time
The time the event ends. This will be displayed on the event popup and Upcoming Events list.
#### Description
This is where you can add more info about your event. Add the details that wouldn't fit in the title but that people will want to know before signing up or arriving to the event.
#### Registration Link
If an event requires a customer to register for an event beforehand, a registration link can be included to direct them to the event signup. The link will open in a new tab on the customer's device.
## Holidays and Altered Hours
This section is for instances where you need to be closed on a day the store would normally be open or change the hours of operation for a day. These are separate from Events but also appear on the calendar and Upcoming Events list.
### Holidays and Altered Hours Fields
- Title
- Date
- Closed All Day?
- Open (optional)
- Closed (optional)
#### Title
The title of the occasion for which you are closed or have adjusted your hours. The title appears on the event popup and the Upcoming Events list.
#### Date
The date of the occasion. This tells the site where to place the event on the calendar and how to organize the event in the Upcoming Events list. The date is displayed on the event popup and in the Upcoming Events list.
#### Closed All Day?
A toggle to indicate if your store will be closed for the entire day or not. Toggled on (to the right) means customers should expect your store to be closed for the day. This displays as "Closed" on the calendar. Toggled off (to the left) means that you will be open with adjusted hours. This displays as "Open" on the calendar. If toggled off, it is important to fill in the Open and Closed fields to let customers know what hours you will be open. 
#### Open
The time your store will open that day. This will be displayed on the event popup and in the Upcoming Events list.
#### Closed
The time your store will open that day. This will be displayed on the event popup and in the Upcoming Events list.
## Hours
This section is to indicate your stores permanent hours of operation. Displays on the home page.
### Hours Fields
- Open
- Open Time
- Close Time
#### Open
A toggle to indicate if your store will be open that day. Toggled on (to the right) means that the store will be open some time that day. If this is toggled on, it is important to indicate the Open Time and Close Time. Toggled off (to the left) indicates that the store will not be open at any point on that day.
#### Open Time
The time the store will open that day. Displays on the home page if that day's Open toggle is set to on.
#### Close Time
The time the store will close that day. Displays on the home page if that day's Open toggle is set to on.
## Offerings
The Offerings section is to provide customers with information on what options they can expect from the store. Displays on the home page.
### Offerings Fields
- Title
- Description
- Display Order
- Link Text (optional)
- URL (optional)
- Image (optional)
- Image Alt Text (optional)
#### Title
The title of the item on offer. Should be descriptive yet concise.
#### Description
Go into further details about what the customer can expect from this type of product or feature.
#### Display Order
The order in which this item should appear in the Offerings sections. Items are displayed in decending order.
#### Link Text
You can choose to direct customers to an internal or external site. This field is to briefly indicate where the link will take them.
#### URL
To link to an external site, a full URL must be provided (e.g., https://www.google.com). To link to an internal page, only the subpage (the portion of the URL after w30.wine/) should be provided (e.g., wine-club).
#### Image
Here, you are given the option to upload an image or specify an image URL to help customers visualize what they can expect. Avoid using overly large images since they can slow down page load times. Crop images down to only what is necessary, centering the subject, and compress images down to a reasonable size (100KB-200KB).
#### Image Alt Text
If an image is used, adding alt text is an important accessibility feature. Alt text is used by screen readers to let visually impaired customers know what the image conveys. The text should be one sentence that describes the image.
